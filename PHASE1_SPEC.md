# Cash App Advocate Workflow Guidance Tool

## Phase 1 Functional Specification

### Objective

Provide advocates with guided workflows that standardize customer issue resolution while surfacing relevant actions, knowledge articles, tools, and scripting.

---

## Navigation

### Dashboard

Purpose:
Provide high-level access to workflow resources.

Contents:
- Workflow Count
- Knowledge Article Count
- Tool Count
- Version Information

---

### Workflow Guidance

Purpose:
Guide advocates through supported workflows.

Features:
- Workflow Intake
- Dynamic Decision Tree
- Previous Step
- Reset Workflow
- Cancel Workflow
- Progress Tracker
- Completion Screen

---

### Knowledge Hub

Purpose:
Centralized access to workflow-related knowledge articles.

Contents:
- Account Access Recovery
- Direct Deposit
- P2P Misdirected Payment
- Unauthorized Cash Card
- Account Takeover

---

## Shared Verification Engine

Flow:

Tap To Confirm?

YES → Verified

NO → Manual Verification

PII Verified?

YES → Continue Workflow

NO → Unable To Verify → Call Closure

---

## Workflow Stages

1. Verification
2. Investigation
3. Assessment
4. Resolution
5. Call Closure
