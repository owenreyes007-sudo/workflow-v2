# Cash App Advocate Workflow Guidance Tool

## Overview

The Cash App Advocate Workflow Guidance Tool is a decision-support application designed to guide advocates through complex customer support workflows while ensuring consistency, compliance, and adherence to operational procedures.

The tool provides structured decision trees, recommended actions, knowledge article references, tool guidance, and advocate scripting to assist with issue resolution.

---

## Purpose

This project aims to:

- Standardize advocate handling processes
- Reduce navigation time across multiple knowledge articles
- Improve workflow compliance
- Provide step-by-step decision support
- Increase resolution consistency
- Reduce onboarding and nesting ramp time
